## H5P Generator
This project is used to to generate h5p archives based on contents from [h5p.com](https://h5p.com)

## Features
 - [ ] Load h5p archives
 - [ ] Unzip the archives
 - [ ] Customize the content
 - [ ] Zip the archive and save them under a new file.
 - [ ] Preview the generated h5p content
